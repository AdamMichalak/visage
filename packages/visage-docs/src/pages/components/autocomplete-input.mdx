# AutocompleteInput

## Examples

### Default

```jsx live=true
<AutocompleteInput
  id="autocomplete-field"
  onChange={value => console.log('automatic-field', value)}
  options={() => Promise.resolve(['a', 'b', 'c', 'd'])}
  placeholder="Enter a value"
/>
```

### Invalid

