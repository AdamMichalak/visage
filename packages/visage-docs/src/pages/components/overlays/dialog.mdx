# Dialog

```jsx live=true
<WithState defaultValue={false}>
  {(show, setShow) => {
    if (!show) {
      return (
        <Button onClick={() => setShow(true)} type="button">
          Show dialog
        </Button>
      );
    }

    return (
      <Dialog
        id="dialog-unique-id"
        label="Closable modal"
        onClose={e => {
          console.log(e);
          setShow(false);
        }}
      >
        Content
      </Dialog>
    );
  }}
</WithState>
```
