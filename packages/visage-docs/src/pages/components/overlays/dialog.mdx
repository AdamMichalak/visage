# Dialog

## Import

```js transpile=false
import { Dialog } from '@byteclaw/visage';
```

## Examples

```jsx live=true
<WithState defaultValue={false}>
  {(show, setShow) => {
    if (!show) {
      return (
        <Button onClick={() => setShow(true)} type="button">
          Show dialog
        </Button>
      );
    }

    return (
      <Dialog
        id="dialog-unique-id"
        label="Closable modal"
        onClose={e => {
          console.log(e);
          setShow(false);
        }}
      >
        <form action="">
          <label htmlFor="searchable">Searchable select value</label>
          <Select
            id="searchable"
            onChange={value => console.log(value)}
            options={() => Promise.resolve(['a', 'b', 'c', 'd'])}
            placeholder="Select a value"
            searchable
          />
        </form>
        Content
      </Dialog>
    );
  }}
</WithState>
```
