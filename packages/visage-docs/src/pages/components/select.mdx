# Select

## Examples

### Non filterable Select with simple array

If items are passed as an array, filtering is not available

```jsx live=true
<Fragment>
  <Select
    onChange={value => console.log(value)}
    options={['A', 'B', 'C', 'D', 'E', 'F']}
    placeholder="Select a value"
  />
</Fragment>
```

### Filterable Select with simple array

If items are passed as an array, filtering is not available

```jsx live=true
<Fragment>
  <Select
    filterable
    onChange={value => console.log(value)}
    options={['A', 'B', 'C', 'D', 'E', 'F']}
    placeholder="Select a value"
  />
</Fragment>
```

### Read only Select with simple array

Read only select can be focused but can't be manipulated

```jsx live=true
<Fragment>
  <Select
    onChange={value => console.log(value)}
    options={['A', 'B', 'C', 'D', 'E', 'F']}
    readOnly
    placeholder="Select a value"
  />
</Fragment>
```

### Read only Select

Read only select can be focused but can't be manipulated

```jsx live=true
<Fragment>
  <Select
    onChange={value => console.log(value)}
    options={['A', 'B', 'C', 'D', 'E', 'F']}
    readOnly
    placeholder="Select a value"
    defaultValue="A"
  />
</Fragment>
```

### Non filterable Select with simple a function

```jsx live=true
<Fragment>
  <Select
    onChange={value => console.log(value)}
    options={filter =>
      new Promise(resolve =>
        setTimeout(
          resolve,
          1000,
          ['A', 'B', 'C', 'D', 'E', 'F'].filter(val =>
            !filter ? true : val === filter,
          ),
        ),
      )
    }
    placeholder="Select a value"
  />
</Fragment>
```
