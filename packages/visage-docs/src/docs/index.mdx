# Visage

Visage is a design system based set of UI components for [React](https://reactjs.org/) applications. It's goal is to provide you building blocks 
for rapid UI development and bridge the gap between designer and developer, by establishing a common language.
Under the hood Visage uses [Emotion](http://emotion.sh) CSS-in-JS library.
